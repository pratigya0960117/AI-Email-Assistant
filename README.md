# AI-Email-Assistant
# ✉️ AI Email Assistant (Chrome Extension)

An AI-powered email assistant that generates professional email drafts directly inside Gmail using a Chrome Extension.

---

## 🚀 Features

- ✨ Generate professional emails instantly
- 🎯 Supports multiple tones (formal, casual, apology, request)
- 🔗 Integrated with Gmail using Chrome Extension
- ⚡ Real-time response using Gemini API
- 🌐 Full-stack architecture (React + Spring Boot)

---

## 🧠 Tech Stack

- Frontend: React (Vite), Material UI
- Backend: Spring Boot (Java)
- AI Model: Gemini API
- Extension: Chrome Extension (Manifest v3)
- Tools: Postman, IntelliJ IDEA, VS Code
